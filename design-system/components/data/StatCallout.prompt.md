Large-format metric for ACV figures, market sizing, dashboard KPIs. The IBM Plex Serif numeral is the point — at 56px it reads as a confident, editorial figure rather than cold telemetry.

```jsx
<StatCallout value="2.4" unit="M" label="Addressable accounts" />
<StatCallout value="94" unit="%" label="Bets laddering to north star" trend="+6%" />
<StatCallout value="18" label="Connected sources" onDark />
```

Number is Serif Navy (white on dark via `onDark`); the data label beneath is Mono. No border — whitespace separates it.
