Compact pill for status, counts, and metadata. Tinted background + matching text; the label always carries the meaning so the badge never relies on color alone.

```jsx
<Badge>Draft</Badge>
<Badge tone="success">Coherent</Badge>
<Badge tone="warning">Needs review</Badge>
<Badge tone="info">First-party</Badge>
```

Pill radius, Sans SemiBold 12px. Pass `icon` for a leading dot/glyph.
