Inline banner for informational, success, warning, and error messages. Uses a 4px left accent border over an 8% tint — never a saturated full-color fill, which would feel aggressive.

```jsx
<Alert type="info" title="Source connected">Amplitude will begin syncing within the hour.</Alert>
<Alert type="warning">Two bets conflict in the current plan. Review before lock.</Alert>
```

Pair every state with its title/text — meaning is never carried by color alone.
